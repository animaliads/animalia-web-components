## Desenvolvimento

### Instalação

Você pode instalar o componente de forma isolada, conforme o exemplo abaixo:

```
npm install @animaliads/ani-button
```

Ou instalar a biblioteca inteira de componentes:

```
npm install @animaliads/web-components
```

### Como utilizar

Para utilizar o web component na sua aplicação você precisa importar o arquivo principal:

```html
<script src="./node_modules/@animaliads/ani-button"></script>
```

Mas caso você estiver utilizando algum framework, pode ser necessário seguir a documentação do próprio
componente para adicionar o web component. Ou apenas importar o arquivo do componente
em um arquivo JS, por exemplo:

```javascript
import '@animaliads/ani-button';
```

Depois dessa configuração, basta adicionar a tag `ani-button` no seu código:

```html
<ani-button>Hello world!<ani-button></ani-button></ani-button>
```

Para customizar o tema do componente, você pode estar utilizando as propriedades customizáveis
da seguinte forma:

```css
ani-button {
  --background: red;
  --line-height: 100%;
  --border-radius: 3px;
}
```

Para saber quais são as propriedades disponível, consulte [propriedades customizar do botão](#propriedades-customizáveis).
