## Propriedades customizáveis

O componente de botão possui propriedades customizáveis que podem variar de acordo com o tema selecionado. As propriedades customizáveis podem ser alteradas para cada produto, de acordo com os tokens de marca e tokens globais. Com isso, garantimos uma consistência no Design System, sem perder a identidade de cada marca.

O botão possui estados e se modifica de acordo com a interação do usuário. São eles: Default, Hover, Focus e Disabled.

### Button

| Propriedade               | Descrição                                                                                                   | Valor padrão                  |
| ------------------------- | ----------------------------------------------------------------------------------------------------------- | ----------------------------- |
| `--font-family`           | família tipográfica usada na label                                                                          | $font-family-text             |
| `--font-size`             | tamanho da fonte na label                                                                                   | $font-size-default            |
| `--font-weight`           | peso da fonte na label                                                                                      | $font-weight-bold             |
| `--line-height`           | altura da linha da label                                                                                    | $line-height-none             |
| `--border-radius`         | arredondamento da borda                                                                                     | $border-radius-md             |
| `--border-width`          | espessura da borda                                                                                          | $border-width-md              |
| `--padding`               | espaço interno entre a label e a borda do background (padding)                                              | 0, 1em                        |
| `--outline-color-focused` | cor do outline do focus                                                                                     | $color-action-focus           |
| `--color`                 | cor principal que será aplicado internamento no botão de acordo com o tipo (primary, secondary ou tertiray) | $color-action-default         |
| `--color-disabled`        | cor principal no estado disabled                                                                            | $color-action-disabled        |
| `--color-pressed`         | cor principal no estado pressed                                                                             | $color-action-pressed         |
| `--color-danger`          | cor principal quando o componente estiver danger                                                            | $color-feedback-negative-dark |

### Primary

| Propriedade              | Descrição                                                           | Valor padrão                     |
| ------------------------ | ------------------------------------------------------------------- | -------------------------------- |
| `--text-color`           | cor da label                                                        | $color-neutral-light-00          |
| `--shadow`               | sombra do background                                                | $shadow-none                     |
| `--color-hover`          | cor do background estado hover                                      | $color-action-hover              |
| `--text-color-disabled`  | cor da label no estado disabled                                     | $color-neutral-dark-70           |
| `--text-color-danger`    | cor da label quando o componente estiver danger                     | $color-neutral-light-00          |
| `--color-danger-hover`   | cor do background estado hover quando o componente estiver danger   | $color-feedback-negative-darker  |
| `--color-danger-pressed` | cor do background estado pressed quando o componente estiver danger | $color-feedback-negative-darkest |
