## Propriedades customizáveis

O componente de modal possui propriedades customizáveis, que possibilitam fazer alterações, de acordo com a marca ou o tema a ser utilizado. Com isso, garantimos uma consistência ao Design System, sem perder a identidade de cada marca.
O modal possui estados e se modifica de acordo com o tipo do modal. Os seus estados são: Normal, Hover, Focused e Selected.

### Dialog

| Propriedade       | Descrição               | Valor padrão            |
| ----------------- | ----------------------- | ----------------------- |
| `--border-radius` | arredondamento da borda | $border-radius-md       |
| `--border-width`  | espessura da borda      | $border-width-sm        |
| `--border-color`  | cor da borda            | $color-neutral-light-20 |
| `--background`    | cor de fundo do modal   | $color-neutral-light-00 |
| `--shadow`        | sombra do background    | $shadow-md              |

### Overlay

| Propriedade         | Descrição               | Valor padrão |
| ------------------- | ----------------------- | ------------ |
| `--color-overlay`   | cor de fundo do overlay |              |
| `--opacity-overlay` | opacidade do overlay    |              |

### Header
