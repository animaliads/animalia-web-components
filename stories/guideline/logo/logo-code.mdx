## Desenvolvimento

A logo deverá ser envolvida em um link para fazer redirecionamento para a página inicial do site.
Também deverá conter no atributo `alt` a informação da função desta imagem, que no caso é um link para a página inicial,
como no exemplo abaixo:

```html
<a href="https://animaliads.io/">
  <img src="logo.svg" alt="Animalia DS home" />
</a>
```

Para mais informações, você pode consultar o [Tutorial de imagens feito pela W3C](https://www.w3.org/WAI/tutorials/images/functional/).
